# Advanced

This folder focuses on production-relevant Go patterns:

1. `01_goroutines_channels/`
2. `02_context_timeout/`
3. `03_mutex_shared_state/`
4. `04_reflect_unsafe_cgo/`

These examples are still small, but they introduce the core building blocks used in concurrent Go services.

Run an example directory with `go run`, for example:

```bash
go run ./advanced/01_goroutines_channels
```
