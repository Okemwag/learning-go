# Basics

This folder covers first-step Go concepts. Each topic lives in its own folder with a runnable `main.go` example and a matching `notes.md` file.

Suggested order:

1. `01_hello/`
2. `02_types_zero_values/`
3. `03_variables_constants/`
4. `04_arrays/`
5. `05_slices/`
6. `06_strings_runes_bytes/`
7. `07_maps/`
8. `08_structs/`
9. `09_blocks_shadowing_if/`
10. `10_for_loops/`
11. `11_switch_goto/`
12. `12_functions/`
13. `13_pointers_memory/`

Run examples one file at a time:

```bash
go run ./basics/01_hello
```
